# 글로벌 매크로 → 한국 주식 분석 룰북 v1.0

> LLM이 시그널 생성·종목 분석 시 이 룰북을 컨텍스트로 참조한다.
> 변경 시 버전을 올리고, 회고 결과를 부록에 누적 기록한다.
> 이 문서는 *예측 가이드*가 아니라 *분석 프레임워크*다.

## 0. 사용 원칙

- 모든 시그널은 **`{관찰된 사실, 적용한 룰, 점수, 출처}`** 구조로 출력한다.
- 룰북 전체가 캐시 prefix에 들어가도록 *시스템 프롬프트 다음 자리*에 주입한다 (Prompt Caching, 1시간 TTL).
- 분기당 1회 회고에서 검증되지 않은 룰은 폐기한다.

## 1. 한국 주식에 영향을 주는 인과 사슬 Top 8

| 변수 | 한국 주식 영향 경로 | 시간 우위 |
|---|---|---|
| 미국 시장 (S&P500, 나스닥, VIX) | 미국 종가 → 한국 갭 (상관계수 약 0.7+) | ★★★ 매우 큼 (마감 후 4시간) |
| 달러/원 환율 (USDKRW, DXY) | 환율 ↑ → 외국인 매도 → 코스피 ↓ (역상관). 수출주는 환율 ↑ 단기 호재 | 실시간 |
| 미국 금리 (10Y, Fed 기준금리) | 금리 ↑ → 성장주(IT) 타격. FOMC 직후 변동성 폭증 | 일정 사전 공지 |
| 반도체 사이클 | 코스피의 25%+ 비중. TSMC/Nvidia 실적, 메모리 가격이 직격 | 발표일 사전 공지 |
| 중국 경제 (PMI, 위안화) | 한국 수출 1위국. 중국 경기 ↓ → 한국 수출주 ↓ | 매월 PMI |
| 유가 (WTI, 브렌트) | 원유 100% 수입. 정유주(호재) vs 항공·운송(악재) — 섹터별 다름 | 실시간 |
| 지정학 (중동·우크라·한반도) | 단기 충격형. 변동성 확대 | 사후만 가능 |
| 국내 변수 (한은 금리, 수급, DART) | 직접 효과. 영업일 기준 | 사전·실시간 |

## 2. 정보 수집 점수 (3축, 각 0~5점)

| 축 | 5점 (강) | 3점 (중) | 1점 (약) |
|---|---|---|---|
| 영향 경로 | 인과 사슬 명확 + 위 8개 변수와 직접 연결 | 간접 영향 | 경로 불명확 |
| 시간 우위 | 1차 정보 + 한국 장 개장 전 4시간 | 같은 날 장 중 | 사후 정보 |
| 영향 크기 | 코스피 평균 ±2% 이상 변동 전례 | ±0.5~1% | ±0.5% 미만 |

**임계값**:
- 합산 ≥ 9 → 수집 + RAG 저장
- 합산 ≥ 12 → 시그널 후보 (요약기 단계로)
- 합산 ≥ 13 + 다중 출처 동조 → 강시그널 (사용자 알림)

## 3. 출처별 신뢰도 가중치

| 분류 | 출처 예시 | 가중치 |
|---|---|---|
| 1차 공식 자료 | SEC EDGAR, FRED, Fed 공식 발표, 한국 DART, 한은 | 1.0 |
| 메이저 매체 (편집 검증) | Reuters, Bloomberg, WSJ, FT | 0.8 |
| 메이저 매체 (속보·해설) | Yahoo Finance, MarketWatch, CNBC, 연합뉴스 | 0.6 |
| 분석 플랫폼·검증 KOL | Seeking Alpha, 검증된 X 계정 | 0.3 |
| 익명 블로그·일반 SNS | — | 0.1 (또는 제외) |

→ 시그널 합산점수 × 출처 가중치 = 최종 시그널 강도

## 4. 시그널 → 행동 매핑

| 시그널 종류 | 조건 | 권장 행동 |
|---|---|---|
| 강매수 | 합산 ≥ 13 + 다중 출처 + 매수 방향 | 사용자 알림 + 후보 종목 제시 |
| 약매수 | 합산 12 + 단일 출처 | 워치리스트 추가, 알림 없음 |
| 강매도 | 합산 ≥ 13 + 보유 종목 부정 시그널 | 사용자 알림 + 매도 후보 제시 |
| 약매도 | 합산 12 + 보유 종목 약시그널 | 손절선 조정 권유 |
| 관망 | 위 조건 미달 | 알림 없음, RAG 저장만 |

**절대 규칙**: 어떤 시그널도 *자동 주문*하지 않는다. CLAUDE.md 절대 규칙 준수.

## 5. 한국 시간대 운영 (KST)

| 시간대 | 모드 | 작업 |
|---|---|---|
| 05:30~08:50 (미국 마감 후) | **집중 분석** | 밤사이 글로벌 이벤트 종합, 한국 장 개장 시그널 생성 |
| 09:00~15:30 (한국 정규장) | **모니터링** | 실시간 공시·시세 감시, 임계값 돌파 시 알림 |
| 15:30~20:00 (장 마감 후) | **회고** | 그날 시그널 vs 실제 변동 비교, 룰북 후보 갱신 메모 |
| 20:00~05:00 (야간) | **수집** | 글로벌 매크로·미국 시장 모니터링, RAG 적재 |

## 6. 회고 규칙

- **일일 회고** (자동, Batch API): 그날 발생한 시그널 vs 실제 종가 비교 → 결과 DB 저장
- **주간 회고** (일요일, Opus): 한 주 시그널 정확도·실패 패턴 분석 → 룰북 후보 변경안 작성
- **분기 회고** (사용자 + Opus): 누적 회고 기반 룰북 v 올리기

**룰 폐기 기준**: 어떤 룰의 시그널 정확도가 3주 연속 50% 미만이면 폐기 후보.

---

## 7. 절대 금지 (위 본 룰과 별개로)

- 사건 발생 *전* 예측 시도 (불가능)
- 단일 익명 출처 기반 시그널 생성
- 자동 주문 (사용자 승인 단계 필수, CLAUDE.md 절대 규칙)
- 룰북에 없는 *즉흥 판단* — LLM이 룰북 외 근거로 시그널 만들면 거부
